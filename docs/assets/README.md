# Project assets

This directory is intended for public-facing repository assets used in the README and releases.

## Recommended files

- `logo.svg` — project logo
- `banner.png` — repository social/banner image
- `gui-main.png` — main desktop GUI screenshot
- `gui-trace.png` — trace / timeline screenshot
- `gui-permissions.png` — permission center screenshot
- `demo.gif` — short product demo animation

## Suggested capture guidelines

- Use a clean, stable app state
- Prefer 2x retina-quality screenshots
- Keep window chrome visible for authenticity
- Avoid sensitive local paths or personal data in screenshots
- Optimize large images before committing
