# Recommended GitHub Discussions Categories

Use GitHub Discussions for questions, design conversations, and broader collaboration that should not start as issues.

## Suggested categories

- **Announcements**
  - release announcements
  - roadmap updates
  - maintenance notices

- **Q&A**
  - usage questions
  - setup questions
  - troubleshooting that is not yet a confirmed bug

- **Ideas**
  - product ideas
  - UX suggestions
  - architecture proposals before they become scoped work

- **Show and tell**
  - demos
  - screenshots
  - experiments or plugin ideas

## Guidance

- Use **Issues** for actionable bugs and feature work
- Use **Discussions** for open-ended questions and proposals
- Convert discussions into issues when work becomes concrete
