# GUI Project Workspace Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add first-class project workspace support to the Kairox desktop GUI, including project metadata, project-bound sessions, Git/worktree display, archive recovery, and chat/composer UI polish.

**Architecture:** Add durable project metadata and session visibility in `agent-store`, expose lightweight project operations through `agent-core::AppFacade` and `agent-runtime::LocalRuntime`, then wire them through Tauri commands into dedicated Pinia stores and Vue components. Preserve existing `list_sessions` semantics for visible non-project sessions and add separate project/archive commands.

**Tech Stack:** Rust workspace crates (`agent-core`, `agent-store`, `agent-runtime`), Tauri 2 commands with `tauri-specta`, Vue 3 Composition API, Pinia setup stores, Vitest, Playwright E2E mock.

```rust
#[derive(Debug, Clone, sqlx::FromRow)]
pub struct ProjectRow {
    pub project_id: String,
    pub workspace_id: String,
    pub display_name: String,
    pub root_path: String,
    pub created_at: String,
    pub updated_at: String,
    pub removed_at: Option<String>,
    pub sort_order: i64,
    pub expanded: bool,
}

#[derive(Clone)]
pub struct ProjectMetaRepository {
    pool: sqlx::SqlitePool,
}

impl ProjectMetaRepository {
    pub fn new(pool: sqlx::SqlitePool) -> Self {
        Self { pool }
    }

    pub async fn create_project(
        &self,
        workspace_id: &str,
        display_name: &str,
        root_path: &str,
        sort_order: i64,
    ) -> crate::Result<ProjectRow> {
        let now = chrono::Utc::now().to_rfc3339();
        let project_id = uuid::Uuid::new_v4().to_string();
        sqlx::query(
            "INSERT INTO kairox_projects
             (project_id, workspace_id, display_name, root_path, created_at, updated_at, sort_order, expanded)
             VALUES (?1, ?2, ?3, ?4, ?5, ?5, ?6, 1)",
        )
        .bind(&project_id)
        .bind(workspace_id)
        .bind(display_name)
        .bind(root_path)
        .bind(&now)
        .bind(sort_order)
        .execute(&self.pool)
        .await?;

        self.get_project(&project_id).await
    }

    pub async fn get_project(&self, project_id: &str) -> crate::Result<ProjectRow> {
        sqlx::query_as::<_, ProjectRow>(
            "SELECT project_id, workspace_id, display_name, root_path, created_at, updated_at,
                    removed_at, sort_order, expanded
             FROM kairox_projects WHERE project_id = ?1",
        )
        .bind(project_id)
        .fetch_one(&self.pool)
        .await
        .map_err(crate::StoreError::from)
    }
}
```

```rust
pub async fn get_git_status(path: &str) -> agent_core::ProjectGitStatus {
    let worktree_path = path.to_string();
    if !std::path::Path::new(path).exists() {
        return agent_core::ProjectGitStatus {
            kind: agent_core::ProjectGitStatusKind::MissingPath,
            branch: None,
            worktree_path,
            message: Some("path does not exist".into()),
        };
    }
    if !std::path::Path::new(path).join(".git").exists() {
        return agent_core::ProjectGitStatus {
            kind: agent_core::ProjectGitStatusKind::NotInitialized,
            branch: None,
            worktree_path,
            message: None,
        };
    }

    let branch = tokio::process::Command::new("git")
        .arg("-C")
        .arg(path)
        .arg("branch")
        .arg("--show-current")
        .output()
        .await;

    match branch {
        Ok(output) if output.status.success() => {
            let branch = String::from_utf8_lossy(&output.stdout).trim().to_string();
            agent_core::ProjectGitStatus {
                kind: if branch.is_empty() {
                    agent_core::ProjectGitStatusKind::Detached
                } else {
                    agent_core::ProjectGitStatusKind::Clean
                },
                branch: (!branch.is_empty()).then_some(branch),
                worktree_path,
                message: None,
            }
        }
        Ok(output) => agent_core::ProjectGitStatus {
            kind: agent_core::ProjectGitStatusKind::Error,
            branch: None,
            worktree_path,
            message: Some(String::from_utf8_lossy(&output.stderr).to_string()),
        },
        Err(error) => agent_core::ProjectGitStatus {
            kind: agent_core::ProjectGitStatusKind::Error,
            branch: None,
            worktree_path,
            message: Some(error.to_string()),
        },
    }
}
```

```rust
pub async fn read_project_instruction_summary(root_path: &std::path::Path) -> agent_core::ProjectInstructionSummary {
    let candidates = [
        "AGENTS.md",
        "CLAUDE.md",
        ".cursorrules",
        "GEMINI.md",
        ".windsurfrules",
        "README.md",
        "README.zh-CN.md",
    ];
    let mut source_paths = Vec::new();
    let mut warning = None;

    for candidate in candidates {
        let path = root_path.join(candidate);
        match tokio::fs::metadata(&path).await {
            Ok(metadata) if metadata.is_file() => source_paths.push(path.display().to_string()),
            Ok(_) => {}
            Err(error) if error.kind() == std::io::ErrorKind::NotFound => {}
            Err(error) => warning = Some(error.to_string()),
        }
    }

    agent_core::ProjectInstructionSummary { source_paths, warning }
}
```
