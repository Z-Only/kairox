# Race-Free Auto-Compaction Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Move the existing auto-compaction trigger from turn START (fire-and-forget `tokio::spawn`) to turn END (serialized through `SessionExecutionRuntime::run_operation`), cover both `SingleStep` and `DagExecution` paths from one call site, and add a `ContextCompactionSkipped` event for the two skip reasons that matter.

**Architecture:** Delete the spawn block in `agent_loop/turn_context.rs`. Add `maybe_schedule_auto_compaction(...)` invoked at the tail of `LocalRuntimeTurnExecutor::execute_turn` for both execution modes. The scheduler reuses `should_trigger_auto_compaction` and `compaction::compact_session`, but enqueues the call through the per-session actor so it never overlaps with an active turn. Skip-with-reason is surfaced as a new `EventPayload::ContextCompactionSkipped { reason, ratio }` variant.

**Tech Stack:** Rust (`agent-core`, `agent-runtime`), Tauri Specta IPC types, Bun + Vue GUI mock.

- [ ] **Step 2: Add `maybe_schedule_auto_compaction` method**

Inside `impl<S, M> LocalRuntimeTurnExecutor<S, M>` (the existing inherent impl block), append:

```rust
/// At the tail of every successful turn, decide whether to enqueue an
/// auto-compaction pass and either schedule it through the session actor
/// (so it serializes after any later `RunTurn`) or emit a
/// `ContextCompactionSkipped` event explaining why no pass was scheduled.
async fn maybe_schedule_auto_compaction(&self, session_id: &SessionId) {
    use agent_core::{
        AgentId, CompactionReason, CompactionSkipReason, DomainEvent, EventPayload,
        PrivacyClassification,
    };

    let threshold = self.config.context.auto_compact_threshold;

    let (last_estimate, model_limits, already_compacting, workspace_id) = {
        let states = self.session_states.lock().await;
        let state = match states.get(&session_id.to_string()) {
            Some(state) => state.clone(),
            None => return,
        };
        let workspace_id = match self.store.load_session(session_id).await {
            Ok(events) => events.first().map(|event| event.workspace_id.clone()),
            Err(_) => None,
        };
        let workspace_id = match workspace_id {
            Some(workspace_id) => workspace_id,
            None => return,
        };
        (
            state.last_estimated_tokens,
            state.model_limits.clone(),
            state.compacting,
            workspace_id,
        )
    };

    let limits = match model_limits {
        Some(limits) => limits,
        None => return,
    };
    let budget = crate::context_budget::build_budget(&limits);
    let usage = agent_core::ContextUsage {
        total_tokens: last_estimate,
        budget_tokens: budget.context_window,
        ..Default::default()
    };
    let ratio = usage.ratio();

    if crate::agent_loop::should_trigger_auto_compaction(
        &usage,
        threshold,
        already_compacting,
    ) {
        let profile_alias = self
            .config
            .context
            .compactor_profile
            .clone()
            .unwrap_or_else(|| {
                let states = std::sync::Arc::clone(&self.session_states);
                match futures::executor::block_on(states.lock())
                    .get(&session_id.to_string())
                    .and_then(|state| state.model_limits.clone())
                {
                    Some(_) => "fake".to_string(),
                    None => "fake".to_string(),
                }
            });
        let store = self.store.clone();
        let event_tx = self.event_tx.clone();
        let model = self.model.clone();
        let session_states = self.session_states.clone();
        let runtime = self.session_execution.clone();
        let workspace_id_clone = workspace_id.clone();
        let session_id_owned = session_id.clone();
        tokio::spawn(async move {
            let _ = runtime
                .run_operation(&session_id_owned, async move {
                    compact_session(
                        &*store,
                        &event_tx,
                        &*model,
                        &profile_alias,
                        &session_states,
                        workspace_id_clone,
                        session_id_owned.clone(),
                        CompactionReason::Threshold { ratio },
                    )
                    .await
                })
                .await;
        });
        return;
    }

    let skip_reason = if already_compacting {
        Some(CompactionSkipReason::AlreadyCompacting)
    } else if threshold >= 1.0 {
        Some(CompactionSkipReason::ThresholdDisabled)
    } else {
        None
    };
    let Some(reason) = skip_reason else {
        return;
    };

    let event = DomainEvent::new(
        workspace_id,
        session_id.clone(),
        AgentId::system(),
        PrivacyClassification::MinimalTrace,
        EventPayload::ContextCompactionSkipped { reason, ratio },
    );
    let _ = crate::event_emitter::append_and_broadcast(&*self.store, &self.event_tx, &event)
        .await;
}
```

```rust
async fn execute_turn(
    &self,
    request: SendMessageRequest,
    cancellation: CancellationToken,
) -> agent_core::Result<()> {
    let session_id = request.session_id.clone();
    let result = match self.execution_mode(&request) {
        ExecutionMode::DagExecution => {
            let executor = self.dag_executor.as_ref().ok_or_else(|| {
                agent_core::CoreError::InvalidState("DAG executor not available".into())
            })?;
            let dag_result = executor
                .execute_with_cancellation(&request, &self.task_graphs, cancellation)
                .await?;
            tracing::info!(
                "DAG execution completed: {} tasks, {} completed, {} failed, {} skipped",
                dag_result.total_tasks,
                dag_result.completed,
                dag_result.failed,
                dag_result.skipped,
            );
            Ok::<_, agent_core::CoreError>(())
        }
        ExecutionMode::SingleStep => {
            let root_path = self.root_path_for_session(&request.session_id).await;
            crate::agent_loop::run_agent_loop(
                crate::agent_loop::AgentLoopDeps {
                    store: &self.store,
                    model: &self.model,
                    event_tx: &self.event_tx,
                    tool_registry: &self.tool_registry,
                    permission_engine: &self.permission_engine,
                    pending_permissions: &self.pending_permissions,
                    memory_store: &self.memory_store,
                    task_graphs: &self.task_graphs,
                    config: &self.config,
                    session_states: &self.session_states,
                    skill_registry: &self.skill_registry,
                    active_skills: &self.active_skills,
                    turn_cancellation: cancellation,
                    root_path,
                },
                &request,
            )
            .await
        }
    };

    if result.is_ok() {
        self.maybe_schedule_auto_compaction(&session_id).await;
    }
    result
}
```
